# VideoTransform 项目运行配置

## Python 环境

- **环境路径**: `D:\Anaconda\envs\script_dev`
- **Python 可执行文件**: `D:\Anaconda\envs\script_dev\python.exe`

## FFmpeg 配置

- **外部路径**: `D:\ffmpeg-master\bin`
- **内置路径**: 项目目录下 `ffmpeg\bin`（优先使用）

## 运行方式

```bash
"D:\Anaconda\envs\script_dev\python.exe" main.py
```

或直接双击 `run.bat`
